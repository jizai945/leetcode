## python3中自定义排序

比如leetcode 179 最大数

```python
import functools

def my_test():
    nums_str = ['92', '123', '929', '920']
    compare = lambda x, y: 1 if x + y < y + x else -1
    # nums_str.sort(cmp = compare)  # python2
    nums_str.sort(key=functools.cmp_to_key(compare)) # python3
    print(nums_str)

if __name__ == '__main__':
    my_test()
```
